<h3 align="center">The Fake Website Detection System is a comprehensive tool that assesses website legitimacy through multiple checks. It generates suspicious URL variants using typo-squatting and homoglyphs, validates each for live status, and performs checks on domain age, SSL certification, and VirusTotalâ€™s malicious site database. Additionally, it scrapes web content to compare similarity with known safe URLs, detecting imitation attempts. A Random Forest Classifier is also applied to evaluate legitimacy based on historical patterns of real and fake sites. Users can scan a URL for similar-looking, potentially fraudulent versions or validate a specific suspicious URL directly, receiving a detailed report of all checks and predictions.</h3>
